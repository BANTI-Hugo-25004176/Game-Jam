# 🎮 Axel

Bienvenue sur le dépôt officiel de **Axel**, un jeu d'action top-down **dieselpunk** développé en un temps record à l'occasion de la game jam **Pixels en Provence** (édition 2026), organisée par le département Informatique de l'IUT d'Aix-Marseille ! 🚀☀️

## ▶️ Jouer maintenant

👉 **[Jouer à Axel sur itch.io](https://androtim.itch.io/axel)** — jouable en navigateur, ou à télécharger (Windows / Linux).

---

## 📝 À propos du projet

*   **Événement :** [Pixels en Provence 2026](https://itch.io/jam/pixels-en-provence) (IUT Aix-Marseille)
*   **Thématique :** lier le numérique, le développement durable et le territoire aixois
*   **Genre :** action / shooter top-down, univers dieselpunk

### L'histoire
Dans une société dieselpunk marquée par la révolution industrielle, vous incarnez **Axel**, le frère jumeau caché de Lexa Von Hohenberg. Tandis que les attaques révolutionnaires contre les nobles se multiplient, Axel se retrouve au cœur du soulèvement. Une **histoire en miroir** avec le jeu parallèle **« Lexa »**.

## 🕹️ Comment jouer ?

| Action | Clavier / Souris | Manette |
|---|---|---|
| Se déplacer | ZQSD / WASD | Stick gauche |
| Viser | Souris | Stick droit |
| Tirer | Clic gauche | R2 / R1 |
| Dash | Espace | A |
| Recharger | R | X |
| Pause | Échap | Start |

*   **Plateformes :** 🌐 Web · 🪟 Windows · 🐧 Linux _(📱 build Android bonus)_
*   Interface **bilingue FR/EN**, contrôles **remappables**.
